# CNC G-Code Generator Pro v4.0

CNC 車床全功能 G-Code 自動產生器

## 功能

### 10 種零件類型
- **螺絲/螺栓** — 外螺紋 + 頭部 + 桿部
- **階梯軸/光軸** — 多段直徑，可選圓角、尾端螺紋
- **套筒/襯套** — 中空管狀件，通孔/盲孔，可選內螺紋
- **牙條** — 全長外螺紋桿
- **銷/圓柱** — 簡單圓柱 + 倒角 + 可選卡環溝
- **管接頭** — 雙端螺紋 + 六角中段 + 通孔
- **法蘭盤/圓盤** — 寬盤 + 中心孔 + 可選凸台
- **溝槽軸** — 軸 + O-ring/卡環溝槽
- **內螺紋件** — 內孔攻牙件
- **錐形件** — 莫氏錐 / 標準錐度 / 自訂錐面

### 4 大控制器
FANUC / OKUMA OSP / Mitsubishi / Siemens

### 7 大牙系 80+ 規格
ISO 公制粗細牙、UNC/UNF 美制、BSP 英制管牙、NPT 錐管牙、ACME 梯形牙、自訂

### 16 種材質
碳鋼、合金鋼、不鏽鋼、鋁合金、黃銅、鈦合金、鎳基合金、工程塑膠

### 12 種標準錐度
莫氏 1-5 號、1:5/1:10/1:20/1:50、7:24 ISO 30/40、自訂

### 其他
- AI 圖紙辨識 (自動判斷零件類型)
- 即時 SVG 零件預覽
- 工時分析 (含鑽孔/搪孔/溝槽時間)
- 自動刀具表 (依零件類型配刀)
- 分段式切斷 (退屑 + 慢速過中心)
- 啄鑽循環 G83
- 搪孔精修
- 內/外螺紋 G76
